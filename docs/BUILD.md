# Building

## Dependencies

Download Dependencies.

- [boost](boost.org) 
- [wangle](https://github.com/facebook/wangle)
- [folly](https://github.com/facebook/folly)
- [proxygen](https://github.com/facebook/proxygen)

# Building

Go to Henhouse project directory and
run...

    mkdir build
    cmake ..
    make -j
    ./src/henhouse/henhouse
